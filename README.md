# Quick Setup


## 1. Install Dependensi
```bash
npm install
```

## 2. Konfigurasi Environment (.env)

1. Buat file `.env` atau `.env.local` di root directory
2. Isi file seperti di bawah ini:

```env
# API Base URL
NEXT_PUBLIC_API_BASE_URL="http://<host>/api/v1"
```

## 3. Jalankan Projek

```bash
npm run dev
```
